# Refactor Proxy

이 스킬의 원본은 `.claude/skills/refactor/SKILL.md`입니다.

이 스킬이 호출되면 다음 순서로 동작합니다.

1. `.claude/skills/refactor/SKILL.md`를 먼저 읽습니다.
2. 원본 스킬의 규칙, 워크플로우, 출력 형식을 그대로 따릅니다.
3. 이 프록시 파일보다 원본 스킬의 지침을 우선합니다.
